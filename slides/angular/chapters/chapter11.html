More Services Integrate the NodeJS Backend

HttpClient

  • injectable class
  • contains methods to perform HTTP requests
  • return type varies according to which signature is called (mainly the values of observe and responseType)

HttpInterceptor

Intercepts HttpRequest and handles them.

  • Use for header manipulation.
  • Most interceptors will transform the outgoing request before passing it to the next interceptor in the chain, by calling next.handle(transformedReq).
